# 🌍 KindMap (Prototype)

KindMap is a frontend prototype of an AI-powered platform designed to connect volunteers with NGOs through intelligent task matching and real-time coordination tools.

This project focuses on designing a clean, scalable UI and user experience for both volunteers and NGOs.

---

## 🚀 What this prototype includes

* Landing page with complete product overview
* Volunteer & NGO flow design
* Task discovery and interaction UI
* Kanban-style task management interface
* Points, badges, and engagement system UI
* Fully responsive modern design

---

## ⚠️ Current Status

This is **not a full-stack application yet**.

* ❌ No backend integration
* ❌ No database
* ❌ No authentication system

👉 This project is focused on **UI/UX and frontend architecture**

---

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript (Vanilla)

---

## 🧠 Purpose of this Project

* To design a scalable volunteer coordination system
* To practice modular frontend architecture
* To simulate a real-world product before backend development

---

## 🔮 Future Scope

* Add PHP + MySQL backend
* Implement authentication (Google OAuth)
* Integrate AI (Gemini API) for task recommendations
* Add real-time data (tasks, users, stats)
* Deploy as full-stack web app
